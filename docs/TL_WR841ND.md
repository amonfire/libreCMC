# TP-WR841ND

## Specs

* SoC : Atheros AR9341
* RAM: 32 MB
* Flash: 4 MB

## Versions Supported

* v5.x
* v8.x
* v9.x
* v10.x <-- may need to be tftp flashed
* v11.x <--
* v12.x <--


DO NOT BUY v13!!! V13 contains a mediatek
chipset, which requires a non-free firmware
