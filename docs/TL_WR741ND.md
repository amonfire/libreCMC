# TP-Link TL-WR741ND

## Specs

* SoC : Atheros AR7240 @ 350 MHz
* RAM: 32 MB
* Flash : 4 MB

## Versions Supported 

* v1 - v2, v4.20 - 4.27 

## Versions **NOT** supported

* v2.1 - v3.1 + v4.0
